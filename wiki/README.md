## Quick Start

1. **Project Setup**

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

2. **Key Directories**

```
/src/
├── entities/     # Business entities
├── features/     # Feature modules
├── shared/       # Shared utilities
└── app/         # Next.js app
```

3. **Development Commands**

```bash
# Run tests
npm run test

# Build for production
npm run build

# Run linting
npm run lint
```
